# goit-advancedjs-hw-01

## Task 1 - Gallery

- Implements an interactive image gallery using SimpleLightbox.
- Dynamically generates gallery items from a predefined image array.
- Clicking an image opens it in a lightbox with captions.

## Task 2 - Form

- Implements a feedback form with email and message fields.
- Saves form data to localStorage as the user types (with throttling).
- Restores saved data on reload and clears storage on successful submit.
